브라우저 API를 어떻게 이해하면 좋을까?

window는 브라우저 환경의 전역 객체(Global Object)로, 브라우저 위에서 동작하는 모든 JavaScript의 최상위 컨텍스트야.

window가 무엇을 담고 있을까?

window.document

window.location

window.history

window.navigator

window.fetch

window.setTimeout

window.console

window.localStorage

정리

평소에 그냥 document, setTimeout, fetch로 쓰는 것들이 사실 전부 window.xxx다. 다만, window를 생략해도 되는 것이었을 뿐

조금 더 정확하게 구분하면,

브라우저는 크게 두 레이어로 이해하면 좋다. 1. JS엔진 : 순수 JS 실행 : 변수 함수 클로저 프로토 타입 2. Web API / window : 브라우저 기능 노출 : DOM, fetch, setTimeout, 이벤트

window는 JS엔진 자체가 아니라, 브라우저가 JS 엔진에게 "이런 기능들을 써도 돼"라고 꽂아준 인터페이스다.

한 줄 요약

"브라우저가 jS에게 window라는 객체를 통해 자신의 기능을 통째로 건네준다"고 이해하면 정확하다.
